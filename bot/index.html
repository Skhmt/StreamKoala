This is a simple page for my bot... for now.

Latest version: 0.4b (15 Dec 2015)

This bot was made in javascript, html, css, and node-webkit (node.js). I used libraries from node-irc, plotly.js, and clipboard.js.

Instructions: download, run the .exe, go to the "Settings" tab, click "Get token", sign in to Twitch, copy the token to include "oauth:", close the pop-up window, paste it into the field next to "Access Token: " and hit "Save token".

Added laptime so you can save, for example, the start of a 24 hr stream that !uptime will count from even if your computer crashes. Minor UI changes all around.

If you're using your bot with a Twitch user other than your primary account, make sure the bot is both a mod and editor in your channel. Go to your Twitch Dashboard to set your bot's account as an editor.

Basic commands:
!uptime - shows the length of time that the bot has been running based on if you want bot uptime, stream uptime, or lap time
!bottime - shows the amount of time the bot has been open
!streamtime - shows the amount of time your stream has been going (if your stream disconnects, this resets)
!laptime - shows the amount of time since you've hit the "lap" button
!updawg - replies with "What's !updawg ?"
!highlight (reasons) - this adds the time stamp to the log with the optional reasons for the highlight. Mods only.
!ht - same as above
!game [game name] - this changes the game. Mods only.
!status [stream status] - this changes the stream title. Mods only.
!permit [username] - allows the user to post links for 60 seconds. Mods only.
!addcom (-ul=[mod/sub/streamer]) [!commandname] [text] - adds a command, can optionally set a required level (mod, sub, or streamer) to use the command. %1 %2 and %3 in the command allows usage of parameters when calling the command. "!addcom !tree %1 and %2 sitting in a tree" followed by "!tree wolfish indigo" will output "wolfish and indigo sitting in a tree". Mods only.
!addcmd - same as above
!delcom [!commandname] - deletes a command. Mods only.
!delcmd - same as above
!quote (id)- plays a random quote or a quote by id.
!addquote [quote author] [quote] - adds a quote.
!delquote [quote id] - deletes a quote.

Spam settings:
Link protection - self explanatory
Word protection - times out based if it finds words in the banned word list
Capital letter protection - only checks english letters, it does count capitals as part of emoticons, so total capitals allowed per post should be relatively high
Symbols protection - symbols are defined as anything besides english letters, numbers, commas, periods, and spaces.
